Getting Started

Integrate AeroLib seamlessly into your runtime ecosystem. The framework uses a performance-optimized web loader configuration to fetch components dynamically.

Loading the UI Library

Acquire the source logic directly via an HTTP payload acquisition engine structure:

Lua Script
local lib = loadstring(game:HttpGet("https://raw.githubusercontent.com/Sealient/AeroLib/refs/heads/main/AeraUI.lua"))()

Initializing the Window Frame

Instantiate your master root wrapper container matrix using the standard .new() constructor framework hook. By default, the layout visibility toggle binds explicitly to Right Control (Enum.KeyCode.RightControl).

Lua Script
local UI = lib.new("MY APEX CORE")

1. Creating Tabs

Tabs isolate individual operation windows into dynamic canvas container states. Elements leverage staggered render timings and elegant hover styling matrices.

Lua Script
local combatTab = UI:CreateTab("Combat")
local visualTab = UI:CreateTab("Visuals")

Note: AeroLib automatically builds and locks operational utility profiles (Settings and Credits profiles) cleanly layouted at the baseline position of your configuration canvas array.

2. Adding Page Elements

Append modules directly onto your instantiated navigation elements using the framework methods detailed below.

Buttons

Interactive matte micro-cards optimized to structure two elements across each responsive layout row.

Lua Script
combatTab:AddButton("Trigger KillAura", function()
    print("KillAura loop engaged.")
end)

Toggles

Decoupled floating toggle structures linked directly to runtime state boolean checks.

Lua Script
combatTab:AddToggle("Auto-Farm Status", false, function(state)
    print("Auto-Farm is now: ", state and "ENABLED" or "DISABLED")
end)

Sliders

Full-width control matrices offering drag tracking and precise direct-click track positioning.

Lua Script
visualTab:AddSlider("FOV Boundary", 30, 120, 70, function(value)
    print("FOV updated to: ", value)
end)

Dropdowns

Fluid array multi-choice overlays featuring layer safety bounding logic masks.

Lua Script
visualTab:AddDropdown("Chams Style", {"Wireframe", "Filled matte", "Thermal Glow"}, "Filled matte", function(selected)
    print("Rendering type selected: ", selected)
end)

Color Pickers

Extended calibration matrices supplying HEX visual streams, precision canvas selectors, and 7-node hue maps.

Lua Script
visualTab:AddColorPicker("ESP Team Color", Color3.fromRGB(168, 85, 247), function(color)
    print("New Color Matrix: ", color)
end)

Keybinds

Advanced listener inputs supporting manual input configurations, dynamically scalable font fields, and clearing methods.

Lua Script
local keybindEx = combatTab:AddKeybind("Aimlock Key", Enum.KeyCode.E, function(chosenKey)
    print("Aimlock key mapped to: " .. chosenKey.Name)
end)

-- Available API controls:

Labels

Basic full-span inline text boxes framework blocks designed to print setup info alerts or parameters details.

Lua Script
combatTab:AddLabel("⚠️ Ensure your FPS is stabilized above 60 before execution.")

3. Global Framework Core Interfaces

Trigger complex global systems using commands bound to the parent interface variable returned during creation setup.

System Notifications

Pops fluid contextual indicators near the baseline interface canvas layer, or pipes text updates via default legacy layers. Types: "Success", "Error", "Info", "Warning", "Premium", "Native".

Lua Script
-- Custom UI Obsidian notification card
UI:Notification("Success", "Configuration protocols loaded cleanly.", 5)

-- Roblox CoreGui system notification fallback
UI:Notification("Native", "Script engine verified successfully.", 4)

Modal Popups

Locks input interaction focal frames, scaling automatically with your variable configurations while sliding out nicely when selection elements fire.

Lua Script
UI:Popup({
    Title = "Execute System Override?",
    Description = "This configuration will rewrite your current local profile settings. Proceed with caution.",
    Options = {
        {
            Title = "Abort",
            Type = "Secondary",
            Callback = function() print("Override cancelled.") end
        },
        {
            Title = "Confirm Override",
            Type = "Danger",
            Callback = function() print("Override authorized.") end
        }
    }
})

Framework Destruction

Cleans the interface stack completely, detaching processing sequences, garbage collecting, and disconnecting environmental triggers.

Lua Script
UI:Destroy()

Complete Example Blueprint

Implement this deployment model configuration block to evaluate your dynamic AeraUI structural asset array features instantly:

Lua Script
local lib = loadstring(game:HttpGet("https://raw.githubusercontent.com/Sealient/AeroLib/refs/heads/main/AeraUI.lua"))()
local UI = lib.new("Vortex Hub")

UI:Notification("Premium", "Welcome back, Premium User!", 4)

local tab1 = UI:CreateTab("Main Actions")

tab1:AddLabel("🌟 Core Parameters")

tab1:AddToggle("God Mode", false, function(toggled)
    print("Godmode state: ", toggled)
end)

tab1:AddSlider("WalkSpeed Mod", 16, 250, 16, function(val)
    workspace.CurrentCamera.FieldOfView = val -- Example execution logic
end)

tab1:AddButton("Dismantle Project Instance", function()
    UI:Popup({
        Title = "Clean Exit?",
        Description = "Are you sure you want to completely unload this UI?",
        Options = {
            { Title = "Cancel", Type = "Secondary" },
            { Title = "Unload Framework", Type = "Danger", Callback = function() UI:Destroy() end }
        }
    })
end)
